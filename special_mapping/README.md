# Special Mapping 特殊映射

## 概述

此目录用于存储需要特殊处理的翻译映射，解决上下文相关的翻译歧义问题。

## 目录结构

```
special_mapping/
├── ProduceItem/
│   ├── pitem_01-2-096-0.json      # 复活琴音的棒冰 - "を"需要翻译为"向"
│   └── [其他特例文件]
├── Character/
│   └── [角色相关特例]
└── README.md                       # 本文档
```

## 使用规则

1. **文件命名**：使用主要键名作为文件名，如 `pitem_01-2-096-0.json`
2. **完整记录**：每个特例文件包含该条目的所有相关键值对
3. **优先级**：special_mapping 具有最高优先级，会覆盖其他所有映射

## 合并优先级

```
special_mapping > todo/new > jp_cn > temp_key_cn(data)
```

## 添加新特例的步骤

1. 在相应的子目录中创建新的JSON文件
2. 从现有翻译文件中提取完整的键值对
3. 确认翻译准确性
4. 更新此README文档记录

## 当前特例列表

### ProduceItem

- **pitem_01-2-096-0.json** - "复活琴音的棒冰"
  - 问题：日文"を"在此上下文中应翻译为"向"而非常见的"会"
  - 解决：完整记录该条目的所有翻译，避免在merge时产生冲突

## 维护注意事项

1. 定期检查特例是否仍然适用
2. 避免滥用特例映射
3. 尽量在jp_cn中处理通用情况
4. 记录特例产生的原因和背景
